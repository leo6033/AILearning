# 概述

LightGBM 是微软开发的一款快速、分布式、高性能的基于决策树的梯度 Boosting 框架。其优势有：

1. 训练效率高
2. 内存使用少
3. 准确率高
4. 支持并行学习
5. 可处理大规模数据
